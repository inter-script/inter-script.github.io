Welcome to InterScript

A new programming language for seamless backend and frontend integration.

About InterScript

InterScript is designed to bridge the gap between backend and frontend development, offering a seamless and efficient way to create web applications.

Features

Easy Integration

Integrate your backend and frontend effortlessly with InterScript.

Open Source

Fully open source, allowing the community to contribute and create libraries.

WebSocket Support

Native WebSocket support for real-time communication.

Installation

To install InterScript, follow the instructions below:

python main.py inter install <library-name>
